# UX Usability Study

This site presents a UX usability study overview for an AI-focused learning module targeted at first-year UX students.

## What’s on `Index.html`

- A navigation bar linking to Overview, Participants, Tasks & Metrics, Findings, and Design Changes.
- A project goal summary explaining the study purpose, which is to assess how well students understand AI in the UX process and navigate the learning module.
- A study snapshot with participant count, round count, session length, and target completion rate.
- A low-fidelity prototype section with a link to the Canva prototype.
- A footer that reinforces the study focus.

## Files

- `Index.html` — the main study overview page content
- `style.css` — the page styling for the index and related pages

## Notes

This page is a simple prototype summary for UX research and is designed to look like a study report rather than a finished product.
